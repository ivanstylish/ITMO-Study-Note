```
public class Box<T> {

```  

`? extends T`: 接受 T 或其子类（上界通配符）。  
`? extends T`: Принимает T или его подклассы (подстановочный знак верхней границы).

`? super T`: принимает T или его родителя (подстановочный знак нижней границы)

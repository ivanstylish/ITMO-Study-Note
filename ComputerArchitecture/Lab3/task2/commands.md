一、 数据移动指令 / Инструкции перемещения данных
lit <value>
中：字面量。将一个立即数值压入数据栈。
俄：Литерал. Помещает непосредственное значение в стек данных. (dataStack.push(<value>))
@p <address>
中：从指定地址读取。将特定内存地址中的值压入数据栈。
俄：Чтение по адресу. Помещает значение из заданного адреса памяти в стек данных. (dataStack.push(mem[<address>]))
@
中：读取。将寄存器 A 所指向的内存地址中的值压入数据栈。
俄：Чтение. Помещает значение из адреса памяти, на который указывает регистр A, в стек данных. (dataStack.push(mem[A]))
@+
中：读取并递增。将寄存器 A 所指向地址的值压入数据栈，然后将 A 的值加 1。
俄：Чтение с инкрементом. Помещает значение по адресу A в стек данных, затем увеличивает A на 1. (dataStack.push(mem[A]); A <- A + 1)
@b
中：读取 B。将寄存器 B 所指向的内存地址中的值压入数据栈。
俄：Чтение B. Помещает значение по адресу регистра B в стек данных. (dataStack.push(mem[B]))
!p <address>
中：存储到指定地址。将数据栈顶的值弹出，存入特定的内存地址。
俄：Сохранение по адресу. Извлекает верхнее значение стека данных и сохраняет его по заданному адресу. (mem[<address>] <- dataStack.pop())
!
中：存储。将数据栈顶的值弹出，存入寄存器 A 所指向的内存地址。
俄：Сохранение. Извлекает верхнее значение стека данных и сохраняет его по адресу регистра A. (mem[A] <- dataStack.pop())
!+
中：存储并递增。将数据栈顶的值弹出存入 A 指向的地址，然后将 A 的值加 1。
俄：Сохранение с инкрементом. Извлекает верхнее значение стека, сохраняет по адресу A, затем увеличивает A на 1. (mem[A] <- dataStack.pop(); A <- A + 1)
!b
中：存储到 B。将数据栈顶的值弹出，存入寄存器 B 所指向的内存地址。
俄：Сохранение в B. Извлекает верхнее значение стека данных и сохраняет его по адресу регистра B. (mem[B] <- dataStack.pop())
a!
中：存储到 A。将数据栈顶的值弹出，赋值给寄存器 A。
俄：Сохранение в A. Извлекает верхнее значение стека данных и присваивает его регистру A. (A <- dataStack.pop())
b!
中：存储到 B。将数据栈顶的值弹出，赋值给寄存器 B。
俄：Сохранение в B. Извлекает верхнее значение стека данных и присваивает его регистру B. (B <- dataStack.pop())
a
中：读取 A。将寄存器 A 的值压入数据栈。
俄：Чтение A. Помещает значение регистра A в стек данных. (dataStack.push(A))
二、 算术指令 / Арифметические инструкции
+
中：加法。将数据栈顶的两个值弹出并相加，将结果压入栈，同时设置进位标志。
俄：Сложение. Извлекает два верхних значения из стека, складывает их, помещает результат в стек и устанавливает флаг переноса.
(注：在扩展算术模式 EAM 下，会额外加上当前的进位标志值)
+*
中：乘法步进。执行乘法的一个微操作步骤。
俄：Шаг умножения. Выполняет один микрошаг операции умножения.
+/
中：除法步进。执行一次恢复余数法的微操作步骤。
俄：Шаг деления. Выполняет один микрошаг алгоритма деления с восстановлением остатка.
2*
中：左移一位。将数据栈顶的值向左移动一位（相当于乘以 2）。
俄：Сдвиг влево на 1 бит. Сдвигает верхнее значение стека влево на один бит (эквивалент умножения на 2). (dataStack.push(dataStack.pop() << 1))
2/
中：右移一位。将数据栈顶的值向右移动一位，保留符号位（算术右移，相当于除以 2）。
俄：Сдвиг вправо на 1 бит. Сдвигает верхнее значение стека вправо на один бит с сохранением знака (эквивалент деления на 2). (dataStack.push(dataStack.pop() >> 1))
inv
中：按位取反。将数据栈顶的值的所有二进制位翻转（0变1，1变0）。
俄：Побитовое инвертирование. Инвертирует все биты верхнего значения в стеке данных (0 становится 1, 1 становится 0). (dataStack.push(~dataStack.pop()))
eam
中：设置扩展算术模式。弹出数据栈顶的值，用于设置 EAM 标志位。
俄：Установка расширенного арифметического режима. Извлекает значение из стека данных и устанавливает флаг EAM. (eam <- dataStack.pop())
三、 位操作指令 / Побитовые инструкции
and
中：按位与。弹出栈顶两值，进行按位 AND 运算后压栈。
俄：Побитовое И. Извлекает два значения, выполняет операцию побитового И и помещает результат в стек. (dataStack.push(dataStack.pop() & dataStack.pop()))
xor
中：按位异或。弹出栈顶两值，进行按位 XOR 运算后压栈。
俄：Побитовое исключающее ИЛИ. Извлекает два значения, выполняет операцию побитового XOR и помещает результат в стек. (dataStack.push(dataStack.pop() ^ dataStack.pop()))
四、 栈操作指令 / Инструкции манипуляции стеком
drop
中：丢弃。移除并丢弃数据栈的最顶部的值。
俄：Сброс. Удаляет и отбрасывает самое верхнее значение из стека данных. (dataStack.pop())
dup
中：复制。复制数据栈的最顶部的值并压入栈（此时栈顶有两个相同的值）。不改变进位标志。
俄：Дублирование. Копирует самое верхнее значение стека данных и помещает его в стек (теперь наверху два одинаковых значения). Не изменяет флаг переноса. (dataStack.push(dataStack.top()))
over
中：交换。交换数据栈最顶部的两个值的位置。
俄：Обмен. Меняет местами два верхних значения в стеке данных. (T <- pop(); S <- pop(); push(T); push(S))
五、 控制流指令 / Инструкции управления потоком
;
中：返回。从子程序返回（将返回栈顶的地址弹出到程序计数器）。
俄：Возврат. Возврат из подпрограммы (извлекает адрес из верха стека возвратов в счетчик программ). (p <- returnStack.pop())
<label>
中：调用。调用指定标签处的子程序（将当前地址压入返回栈，跳转到标签）。
俄：Вызов. Вызов подпрограммы по указанной метке (текущий адрес помещается в стек возвратов, переход к метке). (returnStack.push(p); p <- <label>)
<label> ;
中：跳转。无条件跳转到指定的标签处。
俄：Переход. Безусловный переход к указанной метке. (p <- <label>)
next <label>
中：循环。跳转到指定标签，同时将返回栈顶部的值（R）减 1。如果减 1 后不为 0，继续循环；如果为 0，则退出循环。
俄：Цикл. Переход к указанной метке с одновременным уменьшением верхнего значения стека возвратов (R) на 1. Если не 0 - продолжение цикла, если 0 - выход из цикла.
if <label>
中：条件为0则跳转。如果数据栈顶的值等于 0，则跳转到指定标签（值会被弹出）。
俄：Переход если ноль. Если верхнее значение стека данных равно 0, происходит переход к указанной метке (значение извлекается). (if dataStack.pop() == 0 then p <- <label>)
-if <label>
中：非负则跳转。如果数据栈顶的值大于或等于 0，则跳转到指定标签（值会被弹出）。
俄：Переход если неотрицательное. Если верхнее значение стека данных больше или равно 0, происходит переход к указанной метке (значение извлекается). (if dataStack.pop() >= 0 then p <- <label>)
halt
中：停机。终止处理器的运行。
俄：Остановка. Прекращает работу процессора.
六、 寄存器操作指令（栈间移动） / Инструкции манипуляции регистрами (перемещение между стеками)
r>
中：R入T。将返回栈顶部的值移动（弹出）到数据栈的顶部。
俄：R в T. Перемещает (извлекает) верхнее значение из стека возвратов в верх стека данных. (dataStack.push(returnStack.pop()))
>r
中：T入R。将数据栈顶部的值移动（弹出）到返回栈的顶部。
俄：T в R. Перемещает (извлекает) верхнее значение из стека данных в верх стека возвратов. (returnStack.push(dataStack.pop()))

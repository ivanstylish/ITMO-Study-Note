一、 数据移动指令 / Инструкции перемещения данных
lit <value>  
语法 / Синтаксис: lit <value>  
中文解释: 字面量指令。将一个立即数（常数值）压入数据栈的顶部。  
俄文解释: Литерал. Помещает непосредственное значение (константу) в верх стека данных.  
操作 / Операция: dataStack.push(<value>)  
@p <address>  
语法 / Синтаксис: @p <address>
中文解释: 从绝对地址读取。读取指定内存地址中的 32 位值，并将其压入数据栈。
俄文解释: Чтение по абсолютному адресу. Считывает 32-битное значение из указанного адреса памяти и помещает его в стек данных.
操作 / Операция: dataStack.push(mem[<address>])
@
语法 / Синтаксис: @
中文解释: 读取（基于 A）。读取寄存器 A 所指向的内存地址中的值，压入数据栈。
俄文解释: Чтение (по A). Считывает значение из адреса памяти, на который указывает регистр A, и помещает его в стек данных.
操作 / Операция: dataStack.push(mem[A])
@+
语法 / Синтаксис: @+
中文解释: 读取并递增。读取寄存器 A 指向的内存值压栈，然后将寄存器 A 的值加 1（按字节递增）。
俄文解释: Чтение с инкрементом. Считывает значение по адресу A в стек, затем увеличивает значение регистра A на 1 (инкремент на байт).
操作 / Операция: dataStack.push(mem[A]); A <- A + 1
@b
语法 / Синтаксис: @b
中文解释: 读取（基于 B）。读取寄存器 B 所指向的内存地址中的值，压入数据栈。
俄文解释: Чтение (по B). Считывает значение из адреса памяти, на который указывает регистр B, и помещает его в стек данных.
操作 / Операция: dataStack.push(mem[B])
!p <address>
语法 / Синтаксис: !p <address>
中文解释: 存储到绝对地址。弹出数据栈顶的值，将其存储到指定的内存地址中。
俄文解释: Сохранение по абсолютному адресу. Извлекает верхнее значение стека данных и сохраняет его по указанному адресу памяти.
操作 / Операция: mem[<address>] <- dataStack.pop()
!
语法 / Синтаксис: !
中文解释: 存储（基于 A）。弹出数据栈顶的值，将其存储到寄存器 A 所指向的内存地址中。
俄文解释: Сохранение (по A). Извлекает верхнее значение стека данных и сохраняет его по адресу, указанному регистром A.
操作 / Операция: mem[A] <- dataStack.pop()
!+
语法 / Синтаксис: !+
中文解释: 存储并递增。弹出栈顶值存入 A 指向的地址，然后将寄存器 A 的值加 1。
俄文解释: Сохранение с инкрементом. Извлекает верхнее значение, сохраняет его по адресу A, затем увеличивает значение регистра A на 1.
操作 / Операция: mem[A] <- dataStack.pop(); A <- A + 1
!b
语法 / Синтаксис: !b
中文解释: 存储（基于 B）。弹出数据栈顶的值，存入寄存器 B 所指向的内存地址。
俄文解释: Сохранение (по B). Извлекает верхнее значение стека данных и сохраняет его по адресу, указанному регистром B.
操作 / Операция: mem[B] <- dataStack.pop()
a!
语法 / Синтаксис: a!
中文解释: 弹出至 A 寄存器。弹出数据栈顶的值，赋值给寄存器 A。
俄文解释: Извлечение в регистр A. Извлекает верхнее значение стека данных и присваивает его регистру A.
操作 / Операция: A <- dataStack.pop()
b!
语法 / Синтаксис: b!
中文解释: 弹出至 B 寄存器。弹出数据栈顶的值，赋值给寄存器 B。
俄文解释: Извлечение в регистр B. Извлекает верхнее значение стека данных и присваивает его регистру B.
操作 / Операция: B <- dataStack.pop()
a
语法 / Синтаксис: a
中文解释: 压入 A 寄存器。将寄存器 A 的当前值复制并压入数据栈。
俄文解释: Помещение регистра A в стек. Копирует текущее значение регистра A и помещает его в стек данных.
操作 / Операция: dataStack.push(A)
二、 算术指令 / Арифметические инструкции
+
语法 / Синтаксис: +
中文解释: 加法。弹出栈顶的两个值相加，将和压入栈，并设置进位标志。
俄文解释: Сложение. Извлекает два верхних значения, складывает их, помещает сумму в стек и устанавливает флаг переноса.
操作 / Операция:
(无EAM): dataStack.push(dataStack.pop() + dataStack.pop()), set carry flag
(有EAM): dataStack.push(dataStack.pop() + dataStack.pop() + carry), set carry flag
+*
语法 / Синтаксис: +*
中文解释: 乘法步进。执行硬件乘法的一个微步骤。
俄文解释: Шаг умножения. Выполняет один микрошаг аппаратного умножения.
操作 / Операция: T <- T + (if A[0] then S else 0); A <- A >> 1; if T[0] then A[31] <- 1 else A[31] <- 0; T <- T >> 1
+/
语法 / Синтаксис: +/
中文解释: 除法步进。执行一次恢复余数法的微步骤。
俄文解释: Шаг деления. Выполняет один микрошаг деления с восстановлением остатка.
操作 / Операция:
S <- S << 1
if A[31] then S[0] <- 1
A <- A << 1
T <- T << 1
if S >= mem[B] then S <- S - mem[B]; T[0] <- 1
2*
语法 / Синтаксис: 2*
中文解释: 左移一位。将栈顶值向左移动 1 位（相当于乘以 2）。
俄文解释: Сдвиг влево на 1 бит. Сдвигает верхнее значение влево на 1 бит (эквивалентно умножению на 2).
操作 / Операция: dataStack.push(dataStack.pop() << 1)
2/
语法 / Синтаксис: 2/
中文解释: 算术右移一位。将栈顶值向右移动 1 位，并保留符号位（相当于除以 2）。
俄文解释: Арифметический сдвиг вправо на 1 бит. Сдвигает верхнее значение вправо на 1 бит с сохранением знака (эквивалентно делению на 2).
操作 / Операция: dataStack.push(dataStack.pop() >> 1)
inv
语法 / Синтаксис: inv
中文解释: 按位取反。翻转栈顶值的所有二进制位（0 变 1，1 变 0）。
俄文解释: Побитовое инвертирование. Инвертирует все биты верхнего значения (0 становится 1, 1 становится 0).
操作 / Операция: dataStack.push(~dataStack.pop())
eam
语法 / Синтаксис: eam
中文解释: 设置扩展算术模式。弹出栈顶值，用于开启/关闭 EAM 标志。
俄文解释: Установка расширенного арифметического режима. Извлекает верхнее значение для включения/выключения флага EAM.
操作 / Операция: eam <- dataStack.pop()
三、 位运算指令 / Побитовые инструкции
and
语法 / Синтаксис: and
中文解释: 按位与。弹出栈顶两值，执行 AND 运算后压栈。
俄文解释: Побитовое И. Извлекает два значения, выполняет операцию И и помещает результат в стек.
操作 / Операция: dataStack.push(dataStack.pop() & dataStack.pop())
xor
语法 / Синтаксис: xor
中文解释: 按位异或。弹出栈顶两值，执行 XOR 运算后压栈。
俄文解释: Побитовое исключающее ИЛИ. Извлекает два значения, выполняет операцию XOR и помещает результат в стек.
操作 / Операция: dataStack.push(dataStack.pop() ^ dataStack.pop())
四、 栈操作指令 / Инструкции манипуляции стеком
drop
语法 / Синтаксис: drop
中文解释: 丢弃。移除并丢弃数据栈的最顶部的值。
俄文解释: Сброс. Удаляет и отбрасывает самое верхнее значение из стека данных.
操作 / Операция: dataStack.pop()
dup
语法 / Синтаксис: dup
中文解释: 复制。复制数据栈最顶部的值并压入栈（此时栈顶有两个相同的值）。此操作不改变进位标志。
俄文解释: Дублирование. Копирует самое верхнее значение стека данных и помещает его в стек (теперь наверху два одинаковых значения). Эта операция не изменяет флаг переноса.
操作 / Операция: dataStack.push(dataStack.top()), carry flag does not change
over
语法 / Синтаксис: over
中文解释: 交换。交换数据栈最顶部的两个值的位置。（注：此处遵从给定文档描述，与传统Forth的over定义不同）
俄文解释: Обмен. Меняет местами два верхних значения в стеке данных. (Примечание: описание соответствует данному документу, отличается от классического Forth)
操作 / Операция: T <- dataStack.pop(); S <- dataStack.pop(); dataStack.push(T); dataStack.push(S)
五、 控制流指令 / Инструкции управления потоком
;
语法 / Синтаксис: ;
中文解释: 返回。从子程序返回（将返回栈顶的地址弹出到程序计数器 PC）。
俄文解释: Возврат. Возврат из подпрограммы (адрес из вершины стека возвратов передается в счетчик команд PC).
操作 / Операция: p <- returnStack.pop()
<label>
语法 / Синтаксис: <label>
中文解释: 调用。调用指定标签处的子程序（将下一条指令地址压入返回栈，跳转到标签处）。
俄文解释: Вызов. Вызов подпрограммы по указанной метке (адрес следующей инструкции помещается в стек возвратов, происходит переход к метке).
操作 / Операция: returnStack.push(p); p <- <label>
<label> ;
语法 / Синтаксис: <label> ;
中文解释: 无条件跳转。直接将程序计数器设置为指定标签的地址。
俄文解释: Безусловный переход. Непосредственно устанавливает счетчик команд на адрес указанной метки.
操作 / Операция: p <- <label>
next <label>
语法 / Синтаксис: next <label>
中文解释: 循环。将返回栈顶值 (R) 减 1。如果减 1 后不为 0，跳转到标签；如果为 0，则弹出返回栈退出循环。
俄文解释: Цикл. Уменьшает верхнее значение стека возвратов (R) на 1. Если результат не 0, переход к метке; если 0 - извлечение из стека возвратов и выход из цикла.
操作 / Операция: if R != 0 then R <- R - 1; p <- <label> else returnStack.pop()
if <label>
语法 / Синтаксис: if <label>
中文解释: 条件为0跳转。如果数据栈顶的值等于 0，则跳转到指定标签（值被弹出消耗）。
俄文解释: Переход если ноль. Если верхнее значение стека данных равно 0, происходит переход к указанной метке (значение извлекается и потребляется).
操作 / Операция: if dataStack.pop() == 0 then p <- <label>
-if <label>
语法 / Синтаксис: -if <label>
中文解释: 非负跳转。如果数据栈顶的值大于或等于 0，则跳转到指定标签（值被弹出消耗）。
俄文解释: Переход если неотрицательное. Если верхнее значение стека данных больше или равно 0, происходит переход к указанной метке (значение извлекается).
操作 / Операция: if dataStack.pop() >= 0 then p <- <label>
halt
语法 / Синтаксис: halt
中文解释: 停机。终止处理器的运行。
俄文解释: Остановка. Останавливает работу процессора.
六、 寄存器与栈间操作指令 / Инструкции перемещения между регистрами и стеками
r>
语法 / Синтаксис: r>
中文解释: R 入 T。将返回栈最顶部的值移动（弹出）到数据栈的顶部。
俄文解释: R в T. Перемещает (извлекает) верхнее значение из стека возвратов в верх стека данных.
操作 / Операция: dataStack.push(returnStack.pop())
>r
语法 / Синтаксис: >r
中文解释: T 入 R。将数据栈最顶部的值移动（弹出）到返回栈的顶部。
俄文解释: T в R. Перемещает (извлекает) верхнее значение из стека данных в верх стека возвратов.
操作 / Операция: returnStack.push(dataStack.pop())

1. Что такое решение системы линейных алгебраических уравнений? 线性代数方程组的解是什么？
中文：线性代数方程组的解是指满足所有方程的未知数向量 x，使得 Ax = b 成立。
俄文：Решение СЛАУ — это вектор неизвестных x, при котором выполняется равенство Ax = b для всех уравнений системы.

2. Каковы признаки отсутствия решения системы линейных алгебраических уравнений? 线性代数方程组无解的迹象有哪些？
中文：当增广矩阵的秩大于系数矩阵的秩时，系统无解（即 rank([A|b]) > rank(A)）。
俄文：Система не имеет решения, если ранг расширенной матрицы больше ранга матрицы коэффициентов: rank([A|b]) > rank(A).

3. Назовите достоинства и недостатки прямых методов решения СЛАУ. 直接法求解SLAEs的优点和缺点是什么？ 
中文：
优点：计算步数固定，结果精确（在无舍入误差时），适合小规模系统。
缺点：计算量大（O(n³)），对病态矩阵误差放大明显，占用内存多。
俄文：
Достоинства: фиксированное число операций, точный результат (при отсутствии ошибок округления), подходят для небольших систем.
Недостатки: большая вычислительная сложность (O(n³)), сильное накопление ошибок на плохо обусловленных матрицах, большой расход памяти.

4. Назовите достоинства и недостатки итерационных методов решения СЛАУ. 列举迭代法求解SLAEs的优点和缺点。
中文：
优点：对稀疏矩阵友好，内存占用少，适合大规模系统，可提前停止。
缺点：收敛速度依赖矩阵性质（可能不收敛），迭代次数不可预测，需要选择合适的初始向量和精度。
俄文：
Достоинства: экономия памяти, хорошо работают с разреженными матрицами, подходят для больших систем, можно останавливать итерации по достижении нужной точности.
Недостатки: скорость сходимости сильно зависит от свойств матрицы (может не сходиться), число итераций заранее неизвестно, требуется хороший начальный вектор.

5. Как вычислить определитель матрицы, если решать СЛАУ методом Гаусса? 如果使用高斯方法求解线性方程组，如何计算矩阵的行列式？
中文：在高斯消元过程中，行列式等于主元（对角元）的乘积，再乘以每次行交换时产生的 -1（交换次数为奇数时整体取负）。
俄文：Определитель равен произведению диагональных элементов верхнетреугольной матрицы после прямого хода Гаусса, умноженному на (-1)^k, где k — число перестановок строк.

6. В чём заключается идея метода Гаусса с выбором главного элемента по столбцу? 高斯方法（采用列式枢轴选择）背后的思想是什么？
中文：每次消元时，在当前列的所有未处理行中选择绝对值最大的元素作为主元，并将其所在行交换到主对角线位置，以减少误差放大。
俄文：На каждом шаге прямого хода в текущем столбце выбирается элемент с максимальным модулем (главный элемент), строка с этим элементом переставляется на диагональ, чтобы минимизировать ошибки округления.

7. Что такое сходимость итерационного метода? 迭代法的收敛性是什么？
中文：迭代方法收敛是指：随着迭代次数 k → ∞，迭代向量 x^{(k)} 趋向于精确解 x^（即 lim x^{(k)} = x^）。
俄文：Сходимость итерационного метода означает, что последовательность приближений x^{(k)} при k → ∞ стремится к точному решению x^* системы.

8. Каково достаточное условие сходимости итерационных методов? 迭代方法收敛的充分条件是什么？
中文：最常用的充分条件是矩阵满足严格对角占优，或迭代矩阵 G 的某种范数 ||G|| < 1。
俄文：Наиболее часто используемое достаточное условие — строгое диагональное преобладание или норма итерационной матрицы ||G|| < 1 (в любой подчинённой норме).

9. В чём разница между методом простых итераций и методом Гаусса-Зейделя? 简单迭代法和高斯-赛德尔迭代法有什么区别？
中文：简单迭代法（Jacobi）每次迭代都使用上一轮的全部旧值计算新值；高斯-赛德尔法在计算过程中立即使用已更新的新值（从上到下顺序更新）。
俄文：В методе простых итераций (Якоби) все новые значения вычисляются только по предыдущему вектору целиком. В методе Гаусса-Зейделя новые значения используются сразу же после вычисления (по мере обновления сверху вниз).

10. В чём заключается идея метода Гаусса-Зейделя? 高斯-赛德尔迭代法的原理是什么？
中文：将方程组写成 x = Gx + g 的形式，然后每次迭代时，计算 xᵢ 时立即使用前面已更新的 x₁ 到 x_{i-1}，从而加速收敛。
俄文：Система записывается в виде x = Gx + g, а при вычислении компоненты xᵢ на новой итерации сразу используются уже обновлённые значения x₁, ..., x_{i-1}, что ускоряет сходимость.

11. В каких случаях целесообразно использовать метод Гаусса с выбором главного элемента? 在哪些情况下适合使用高斯方法选择主元素？
中文：当矩阵元素大小差异很大、可能导致主元很小或为零、或矩阵病态（条件数大）时，使用列主元选择可以显著减少舍入误差。
俄文：Когда элементы матрицы сильно различаются по модулю, возможны малые или нулевые главные элементы, или матрица плохо обусловлена — выбор главного элемента по столбцу существенно снижает ошибки округления.

12. Как оценить погрешность решения СЛАУ, полученного методом Гаусса? 如何估计使用高斯方法求解SLAE方程的误差？
中文：常用后验误差估计：计算残差 r = b − Ax 的范数 ||r||，然后误差界为 ||r|| / ||A||（或更精确的后验界）。
俄文：Обычно используют апостериорную оценку: вычисляют норму невязки ||r|| = ||b − Ax||, тогда погрешность ограничена сверху выражением ||r|| / ||A|| (или более точными апостериорными оценками).

13. Какие существуют критерии остановки итерационного процесса? 停止迭代过程的标准是什么？
中文：
相邻两次迭代差值：max |x^{(k)} − x^{(k-1)}| < ε
残差范数：||b − Ax^{(k)}|| < ε
相对误差：||x^{(k)} − x^{(k-1)}|| / ||x^{(k)}|| < ε
俄文：
Норма разности двух последовательных приближений: max |x^{(k)} − x^{(k-1)}| < ε
Норма невязки: ||b − Ax^{(k)}|| < ε
Относительная погрешность: ||x^{(k)} − x^{(k-1)}|| / ||x^{(k)}|| < ε

14. Что означает, если норма преобразованной матрицы ≥ 1? 如果变换后矩阵的范数≥1，这意味着什么？
中文：如果迭代矩阵 G 的范数 ||G|| ≥ 1，则收敛性无法保证（可能发散、慢收敛或振荡）。
俄文：Если норма итерационной матрицы ||G|| ≥ 1, то сходимость не гарантируется (метод может расходиться, сходиться медленно или колебаться).

15. Можно ли применять метод Гаусса, если на главной диагонали есть нулевые элементы? 如果主对角线上没有零元素，是否可以应用高斯方法？
中文：可以，但必须使用主元选择（列主元或全主元），否则会出现除零错误或严重误差放大。
俄文：Можно, но обязательно использовать выбор главного элемента (по столбцу или полным), иначе возникнет деление на ноль или сильное накопление ошибок.

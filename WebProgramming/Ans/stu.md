ResultDTO обычно относится к объекту передачи данных (DTO), предназначенному для инкапсуляции результатов операции или запроса. Он служит простым объектом, переносящим данные между различными уровнями приложения, часто между уровнем сервиса и уровнем представления.  
ResultDTO 通常指的是数据传输对象 (DTO)，它旨在封装操作或查询的结果。它作为一个简单的对象，在应用程序的不同层之间传递数据，通常是在服务层和表示层（例如，Web 控制器）之间  

persist() подготавливает новые сущности для сохранения, в то время как flush() выполняет реальные операции с базой данных, чтобы сделать эти изменения постоянными.  
persist() 用于暂存要持久化的新实体，而 flush() 执行实际的数据库操作以使这些更改永久生效。

@Inject — это аннотация внедрения зависимостей (DI) в разработке на Java и Android. Она используется для объявления конструктора, метода или поля, указывающая фреймворку внедрения зависимостей (например, Dagger, CDI или Spring) автоматически внедрить соответствующую зависимость.  
@Inject 是 Java 和 Android 开发中一种依赖注入 (Dependency Injection, DI) 注解，用于声明一个构造器、方法或字段，指示依赖注入框架（如 Dagger, CDI, 或 Spring）应该自动注入相应的依赖

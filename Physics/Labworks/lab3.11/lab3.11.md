实验手册总结（实验室工作 № 3.11）
实验名称：串联振荡电路中的受迫电磁振荡（Вынужденные электромагнитные колебания в последовательном колебательном контуре）
实验目标（Цель работы）：

研究串联 RLC 振荡电路中的受迫振荡以及电压谐振现象。
绘制谐振曲线，确定谐振频率。
测定电路的有功电阻（R）和品质因数（Q）。

1. 实验原理（Теоретическое введение）
电路结构：外部正弦交流电源（电动势 $  \mathcal{E}(t) = \mathcal{E}_0 \cos \Omega t  $）串联接入电阻 $  R  $、电感 $  L  $ 和电容 $  C  $。
电路满足基尔霍夫电压定律，得到电荷 $  q  $ 的微分方程：
$$\frac{d^2 q}{dt^2} + 2\beta \frac{dq}{dt} + \omega_0^2 q = \frac{\mathcal{E}_0}{L} \cos \Omega t$$
其中：

$  \omega_0 = \frac{1}{\sqrt{LC}}  $ —— 无阻尼固有角频率（汤姆逊公式）。
$  \beta = \frac{R}{2L}  $ —— 阻尼系数。

稳态解（暂态衰减后）为：
$$q(t) = q_0 \cos(\Omega t - \psi), \quad I(t) = I_0 \cos(\Omega t - \varphi)$$
电流幅值和相移：
$$I_0(\Omega) = \frac{\mathcal{E}_0}{\sqrt{R^2 + \left( \Omega L - \frac{1}{\Omega C} \right)^2}}, \quad \varphi(\Omega) = \arctan\left( \frac{\Omega L - \frac{1}{\Omega C}}{R} \right)$$
谐振条件：

电流谐振（$  I_0  $ 最大）：$  \Omega_{I_{\rm res}} = \omega_0 = \frac{1}{\sqrt{LC}}  $（与 $  R  $ 无关）。
电容电压谐振（$  U_C  $ 最大）：$  \Omega_{C_{\rm res}} = \sqrt{\omega_0^2 - 2\beta^2}  $（略低于 $  \omega_0  $）。
电感电压谐振（$  U_L  $ 最大）：$  \Omega_{L_{\rm res}} = \frac{\omega_0}{\sqrt{1 - 2(\beta/\omega_0)^2}}  $（略高于 $  \omega_0  $）。

品质因数 $  Q  $（低阻尼 $  \beta \ll \omega_0  $ 时）：
$$Q = \frac{\omega_0}{\Delta \Omega} = \frac{U_{C,\rm res}}{\mathcal{E}_0} = \frac{1}{R} \sqrt{\frac{L}{C}}$$
$  \Delta \Omega  $ 是谐振曲线半高宽（功率下降一半处）。
矢量图和阻抗分析表明：总阻抗 $  X(\Omega) = \sqrt{R^2 + (\Omega L - 1/(\Omega C))^2}  $，谐振时阻抗最小（纯电阻）。
2. 实验逻辑与流程
实验通过测量电容两端电压 $  U_C  $（输出电压）随驱动频率的变化，来观察电压谐振现象。因为实验直接测量的是 $  U_C  $，所以寻找的是电容电压谐振频率（$  \Omega_{C_{\rm res}}  $）。
实验装置：

函数发生器（FG）提供正弦信号（幅值 4.00 V）。
电感线圈（L，FПЭ-11 模块，具体值向工程师询问）。
可变电容箱（C）。
双通道示波器：Channel I 测输入电压（$  \mathcal{E}  $)，Channel II 测输出电压（$  U_C  $)。

测量流程：

固定 $  C = 0.1\,\mu{\rm F}  $，先理论计算不计 $  R  $ 时的谐振频率 $  f_{\rm расч} = \frac{1}{2\pi\sqrt{LC}}  $。
在 $  f_{\rm min} = f_{\rm расч} - 500\,{\rm Hz}  $ 到 $  f_{\rm max} = f_{\rm расч} + 500\,{\rm Hz}  $ 范围内，步进改变频率，记录输出电压幅值 $  U_C  $（示波器读数），绘制 $  U_C(f)  $ 曲线。
改变电容 $  C  $（依次取 1、3、10、30、100、300 nF），对每个 $  C  $ 固定输入幅值，微调频率直到 $  U_C  $ 达到最大，记录该谐振频率 $  f_{\rm res}  $。
安全规则：不可无人值守、不可同时按多键等。

3. 需要测量的数据（Измеряемые величины）

主要数据（C = 0.1 μF 曲线）：
不同频率 $  f  $ 对应的输出电压幅值 $  U_C  $（多点，步进使 $  U_C  $ 变化不超过示波器一个小格）。
谐振时（$  U_C  $ 最大）的频率 $  f_{\rm res}^{\rm exp}  $ 和电压 $  U_{C,\rm res}  $。
输入电动势幅值 $  \mathcal{E}_0 = 4.00\,{\rm V}  $（固定）。

附加数据（不同电容）：
6 个不同 $  C  $ 值（1~300 nF）各自对应的谐振频率 $  f_{\rm res}(C)  $。
电感 $  L  $ 的标称值（向工程师询问）。


4. 实验测量顺序（Порядок выполнения работы）

组装实验装置（见图7）。函数发生器 FG «АКИП 3409/2» 的第一通道应连接到 ФПЭ-11 模块的左侧端子 PO（用于测量输入电压）。示波器第 I 通道连接到 ФПЭ-11 模块右侧端子 PO（测量输入电压），第 II 通道连接到右侧端子 POY（测量输出电压）。
在电容箱上设置电容值为 0.1 μF。
打开函数发生器和示波器。在函数发生器第一通道上将正弦电压的幅值设置为 4.00 V。示波器设置为同时显示第 I 和第 II 通道信号的模式。
已知电容 C 和电感 L 的值（电感值请向工程师询问），估算不考虑电感有功电阻时的谐振频率 $  f_{\rm расч}  $。
在频率区间 $  f_{\rm min} = f_{\rm расч} - \Delta f  $ 到 $  f_{\rm max} = f_{\rm расч} + \Delta f  $（其中 $  \Delta f = 500  $ Hz）内，平滑改变函数发生器 FG 的频率，测量输出电压的幅值。频率步长应选择为使输出电压幅值变化不超过示波器一个最小刻度。将结果记录到表格中。
依次在电容箱上设置电容值 1、3、10、30、100 和 300 nF，对每个电容值，在输入电压幅值不变的情况下，平滑改变频率，直至输出电压达到最大值。记录得到的谐振频率。
关闭装置前，将装置处于开启状态交给值班工程师。
在带有测量结果的实验报告协议上获得工程师签字。
关闭函数发生器和示波器。

5. 测量结果处理（Обработка результатов измерений）

绘制输出电压幅值随输入频率变化的图像。根据图像确定电容上电压振荡幅值达到最大值时的谐振频率。将直接测量的结果与计算得到的谐振频率进行比较。如果存在显著差异，在实验报告中解释其出现原因。
根据绘制的图像，利用公式 (21) 估算振荡电路的品质因数 $  Q  $。结果记入报告。
利用函数发生器电动势幅值和谐振时电容上的电压值，根据公式 (20) 估算振荡电路的品质因数 $  Q  $。结果记入报告，并与由图像得到的结果进行比较。
绘制谐振频率的平方 $  f_{\rm res}^2  $（或 $  \Omega_{\rm res}^2  $）与电容倒数 $  1/C  $ 的关系图。用直线拟合该图像。根据图像的斜率（角系数），利用以下公式求出电感值：

$$\Omega_{\rm res}^2 = \frac{1}{LC} - \frac{R^2}{4L^2}$$
将求得的电感值与给定的电感值进行比较。

根据图像相对于“0”点的偏移量，以及在第 4 步中求得的电感值，估算振荡电路的有功电阻 $  R  $。
实验报告结果必须包含以下内容（针对电容 $  C = 0.1\,\mu{\rm F}  $）：
计算得到的和实验测得的谐振频率值。
用公式 (20) 和 (21) 求得的振荡电路品质因数 $  Q  $。
振荡电路的有功电阻值以及实验测得的电感值（需注明不确定度/误差）。


注意：仪器显示的是频率 $  f  $（单位：Hz），而公式中使用的是角频率 $  \Omega = 2\pi f  $。

📌 重要知识点总结 / Ключевые знания

  🇨🇳 中文

  1. 多项式逼近法的核心思想

  - 在包含极小点的区间上，用多项式（二次或三次）逼近目标函数
  - 用逼近多项式的极小点作为真实极小点的近似
  - 不需要计算二阶导数

  2. 二次逼近（Powell法）vs 三次逼近（Hermite法）

  ┌────────────┬──────────────────┬─────────────────────────────┐
  │  对比维度  │     二次逼近     │          三次逼近           │
  ├────────────┼──────────────────┼─────────────────────────────┤
  │ 使用信息   │ 3个点的函数值    │ 2个端点的函数值+导数值      │
  ├────────────┼──────────────────┼─────────────────────────────┤
  │ 插值多项式 │ Lagrange抛物线   │ Hermite三次多项式           │
  ├────────────┼──────────────────┼─────────────────────────────┤
  │ 收敛速度   │ 线性~超线性      │ 二次收敛（快得多）          │
  ├────────────┼──────────────────┼─────────────────────────────┤
  │ 停止准则   │ `                │ Δx                          │
  ├────────────┼──────────────────┼─────────────────────────────┤
  │ 危险情况   │ 三点共线→分母为0 │ μ可能越界（需clamp到[0,1]） │
  └────────────┴──────────────────┴─────────────────────────────┘

  3. 三次逼近的核心公式（必须记住）

  [
  z = f'(x_1) + f'(x_2) - 3\frac{f(x_2)-f(x_1)}{x_2-x_1}, \quad
  \omega = \sqrt{z^2 - f'(x_1)f'(x_2)}
  ]
  [
  \mu = \frac{\omega + z - f'(x_1)}{2\omega - f'(x_1) + f'(x_2)}, \quad
  x_0 = x_1 + \mu(x_2 - x_1)
  ]

  4. 单峰条件

  - 条件 ( f'(x_1) \cdot f'(x_2) < 0 ) 保证极小点在区间内部
  - 若条件不满足 → 区间内无极小点 → 需要扩大/移动搜索区间

  5. μ ∈ (0, 1) 的数学证明

  - 这是三次逼近法的关键性质 — 新点永远严格位于当前区间内部
  - 保证算法的稳定性和收敛性

  6. Hermite基函数的四个性质

  - ( H_{00}(0)=1, H_{00}(1)=0, H'{00}(0)=H'{00}(1)=0 )
  - ( H_{10}(0)=0, H_{10}(1)=0, H'{10}(0)=1, H'{10}(1)=0 )
  - ( H_{01}(0)=0, H_{01}(1)=1, H'{01}(0)=H'{01}(1)=0 )
  - ( H_{11}(0)=0, H_{11}(1)=0, H'{11}(0)=0, H'{11}(1)=1 )

  🇷🇺 Русский

  1. Ключевая идея полиномиальной аппроксимации

  - На отрезке, содержащем минимум, целевая функция заменяется полиномом (квадратичным или кубическим)
  - Точка минимума полинома берётся как приближение истинного минимума
  - Вторая производная не требуется

  2. Квадратичная (Пауэлл) vs Кубическая (Эрмит)

  ┌─────────────────────────┬────────────────────────┬────────────────────────────┐
  │         Аспект          │      Квадратичная      │         Кубическая         │
  ├─────────────────────────┼────────────────────────┼────────────────────────────┤
  │ Используемая информация │ 3 значения функции     │ 2 значения + 2 производных │
  ├─────────────────────────┼────────────────────────┼────────────────────────────┤
  │ Полином                 │ Лагранж                │ Эрмит                      │
  ├─────────────────────────┼────────────────────────┼────────────────────────────┤
  │ Скорость сходимости     │ линейная~сверхлинейная │ квадратичная               │
  ├─────────────────────────┼────────────────────────┼────────────────────────────┤
  │ Критерий останова       │ `                      │ Δx                         │
  ├─────────────────────────┼────────────────────────┼────────────────────────────┤
  │ Проблемный случай       │ Три точки на прямой    │ μ может выйти за (0,1)     │
  └─────────────────────────┴────────────────────────┴────────────────────────────┘

  (см. выше — z, ω, μ, x₀)

  4. Условие унимодальности

  - ( f'(x₁)·f'(x₂) < 0 ) гарантирует наличие минимума внутри отрезка
  - Производные на концах имеют разные знаки

  5. Свойство μ ∈ (0, 1)

  - Ключевое математическое свойство: новое приближение всегда внутри текущего отрезка
  - Обеспечивает устойчивость и сходимость алгоритма

  6. Базисные функции Эрмита

  Четыре кубических полинома, удовлетворяющих условиям интерполяции в узлах (см. свойства выше)
